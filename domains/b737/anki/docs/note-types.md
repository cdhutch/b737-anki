# Anki Note Formatting Templates

These templates define standardized visual formatting for limits and systems notes.

---

## 1. Operational Hazard Format

<div style="font-family: sans-serif; line-height:1.35em;">
  <strong>[Primary Requirement]</strong>

  <ul style="margin:8px 0 0 18px;">
    <li>[Condition 1]</li>
    <li>[Condition 2]</li>
    <li>[Condition 3]</li>
  </ul>

  <div style="margin-top:8px; padding:6px 8px; border-left:3px solid #999; color:#444;">
    <strong>Caution:</strong> [Consequence or important reminder]
  </div>
</div>
---

## 2. Boolean Logic Format

<div style="font-family: sans-serif; line-height:1.4em;">
  <strong>[Primary statement]</strong>

  <div style="margin-top:8px; padding:8px; border:1px solid #ccc; border-radius:6px;">
    
    <div style="margin-bottom:6px;">
      • [Condition A]
    </div>

    <div style="text-align:center; font-weight:bold; color:#0033cc; margin:6px 0;">
      AND
    </div>

    <div style="margin-left:18px; padding:6px; border-left:3px solid #999;">
      <div style="font-weight:bold; color:#990000; margin-bottom:4px;">
        (ANY ONE OF THE FOLLOWING)
      </div>

      <div>• [Condition B1]</div>

      <div style="text-align:center; font-weight:bold; color:#cc6600; margin:4px 0;">
        OR
      </div>

      <div>• [Condition B2]</div>
    </div>

  </div>
</div>

## 3. Phase-of-Flight Format

<div style="font-family: sans-serif; line-height:1.4em;">
  <strong>[System or limitation]</strong>

  <div style="margin-top:8px; padding:8px; border:1px solid #ccc; border-radius:6px;">

    <div style="margin-bottom:8px;">
      <strong style="color:#003366;">GROUND / CLIMB / CRUISE</strong>
      <div style="margin-left:12px;">
        • [Requirement or condition]
      </div>
    </div>

    <div>
      <strong style="color:#660000;">DESCENT</strong>
      <div style="margin-left:12px;">
        • [Requirement or condition]
      </div>
    </div>

  </div>
</div>

## 4. Failure / Automation Logic Format

<div style="font-family: sans-serif; line-height:1.4em;">
  <strong>[Failure or system event]</strong>

  <div style="margin-top:8px; padding:8px; border:1px solid #ccc; border-radius:6px;">

    <div style="margin-bottom:6px;">
      <strong>Trigger:</strong> [What happens first]
    </div>

    <div style="margin-bottom:6px;">
      <strong>Automatic Response:</strong>
      <ul style="margin:4px 0 0 18px;">
        <li>[Action 1]</li>
        <li>[Action 2]</li>
        <li>[Action 3]</li>
      </ul>
    </div>

    <div style="margin-top:8px; padding:6px 8px; border-left:3px solid #999; color:#444;">
      <strong>Note:</strong> [Important operational consequence]
    </div>

  </div>
